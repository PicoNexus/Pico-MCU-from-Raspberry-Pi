Larger Option Pads
Added option for double ** to make a string starting with * instead of a special * command.
Added comment if a shorter string replaces a longer string end the shorter string with a NULL char - press [GUI]3x and [ADD] at end of shorter string

Compiled with new Pico SDK 1.5, TinyUSB 2.02, Arduino Pico 3.11, TFTeSPI 2.50, and Adafruit GFX 1.11.5

To install new version of Arduino Pico first delete it from boards manager, then delete the folder C:\Users\Tobias\AppData\Local\Arduino15\packages\rp2040 then close and reopen Arduino IDE and then add the new Arduino Pico Board.
